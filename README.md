# MEDITAÇÃO TRATAKA
Um site minimalista para ajudar na prática de meditação com foco visual.

# Como funciona:
Você escolhe o tempo (5, 10 ou 20 minutos), foca no ponto preto central, e um círculo ao redor mostra o tempo passando discretamente.

# Arquivos do Projeto
* index.html: A estrutura do site.
* style.css: O visual (cores, tamanhos e animações).
* script.js: A lógica (o cronômetro e a matemática do círculo).

# Como usar
1. Baixe a pasta com os arquivos.
2. Dê dois cliques no arquivo index.html.
3. O site vai abrir direto no seu navegador.
